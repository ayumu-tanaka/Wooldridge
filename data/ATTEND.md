# ATTEND データセットの説明

**観測数（Obs）:** 680

## 変数一覧

| 変数名 | 説明 |
|------|------|
| attend | 出席したクラス数（32回中） |
| termgpa | 当該学期のGPA |
| priGPA | 当該学期前の累積GPA |
| ACT | ACTスコア |
| final | 期末試験の点数 |
| atndrte | 授業出席率（%） |
| hwrte | 宿題提出率（%） |
| frosh | 新入生の場合1 |
| soph | 2年生の場合1 |
| skipped | 欠席したクラス数 |
| stndfnl | （期末試験点数 - 平均）/ 標準偏差 |

* * *
