# LOWBRTH データセットの説明

**観測数（Obs）:** 100

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1987 or 1990 |
| lowbrth | perc. births low weight |
| infmort | infant mortality rate |
| afdcprt | # participants in AFDC, 1000s |
| popul | population, 1000s |
| pcinc | per capita income, $ |
| physic | Drs. per 100,000 population |
| afdcprc | percent of pop on AFDC |
| d90 | year == 1990 の場合1 |
| lpcinc | log of pcinc |
| cafdcprc | afdcprcの変化 |
| clpcinc | lpcincの変化 |
| lphysic | log of physic |
| clphysic | lphysicの変化 |
| clowbrth | lowbrthの変化 |
| cinfmort | infmortの変化 |
| afdcpay | avg monthly AFDC payment |
| afdcinc | afdcpay as % pcinc |
| lafdcpay | log of afdcpay |
| clafdcpy | lafdcpayの変化 |
| cafdcinc | afdcincの変化 |
| stateabb | 州の郵便番号 |
| state | name of state |
| beds | # hospital beds, 1000s |
| bedspc | beds per capita |
| lbedspc | bedspcの対数 |
| clbedspc | lbedspcの変化 |
| povrate | % people below poverty line |
| cpovrate | povrateの変化 |
| afdcpsq | afdcperの二乗 |
| cafdcpsq | afdcpsqの変化 |
| physicpc | physicians per capita |
| lphypc | physicpcの対数 |
| clphypc | lphypcの変化 |
| lpopul | populの対数 |
| clpopul | lpopulの変化 |

* * *
