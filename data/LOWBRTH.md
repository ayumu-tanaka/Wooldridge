# LOWBRTH データセットの説明

**観測数（Obs）:** 100

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1987または1990年 |
| lowbrth | 低出生体重児の割合（%） |
| infmort | 乳児死亡率 |
| afdcprt | AFDC参加者数（千人） |
| popul | 人口（千人） |
| pcinc | 一人当たり所得（ドル） |
| physic | 人口10万人当たりの医師数 |
| afdcprc | AFDCに参加する人口割合（%） |
| d90 | year == 1990 の場合1 |
| lpcinc | 一人当たり所得の対数 |
| cafdcprc | afdcprcの変化 |
| clpcinc | lpcincの変化 |
| lphysic | 医師数の対数 |
| clphysic | lphysicの変化 |
| clowbrth | lowbrthの変化 |
| cinfmort | infmortの変化 |
| afdcpay | AFDC月次給付額の平均 |
| afdcinc | 一人当たり所得に占めるAFDC給付の割合（%） |
| lafdcpay | AFDC給付の対数 |
| clafdcpy | lafdcpayの変化 |
| cafdcinc | afdcincの変化 |
| stateabb | 州の郵便番号 |
| state | 州名 |
| beds | 病院ベッド数（千台） |
| bedspc | 一人当たりベッド数 |
| lbedspc | bedspcの対数 |
| clbedspc | lbedspcの変化 |
| povrate | 貧困線以下の人口割合（%） |
| cpovrate | povrateの変化 |
| afdcpsq | afdcperの二乗 |
| cafdcpsq | afdcpsqの変化 |
| physicpc | 一人当たりの医師数 |
| lphypc | physicpcの対数 |
| clphypc | lphypcの変化 |
| lpopul | populの対数 |
| clpopul | lpopulの変化 |

* * *

**出典**: [http://fmwww.bc.edu/ec-p/data/wooldridge/lowbrth.des](http://fmwww.bc.edu/ec-p/data/wooldridge/lowbrth.des)
