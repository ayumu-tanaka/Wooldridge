# RDCHEM データセットの説明

**観測数（Obs）:** 32

## 変数一覧

| 変数名 | 説明 |
|------|------|
| rd | 研究開発費（百万ドル） |
| sales | 企業売上高（百万ドル） |
| profits | 利益（百万ドル） |
| rdintens | 研究開発費（売上高比、%） |
| profmarg | 売上高利益率（%） |
| salessq | salesの二乗 |
| lsales | salesの対数 |
| lrd | rdの対数 |
