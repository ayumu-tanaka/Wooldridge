# GPA3 データセットの説明

**観測数（Obs）:** 732

## 変数一覧

| 変数名 | 説明 |
|------|------|
| term | 学期（1=秋、2=春） |
| sat | SATスコア |
| tothrs | 学期前の総単位時間 |
| cumgpa | 累積GPA |
| season | in season の場合1 |
| frstsem | student's 1st semester の場合1 |
| crsgpa | 加重コースGPA |
| verbmath | SAT口語÷数学の比率 |
| trmgpa | 学期GPA |
| hssize | 高校卒業クラスの規模 |
| hsrank | 高校クラスでの順位 |
| id | 学生識別子 |
| spring | spring term の場合1 |
| female | female の場合1 |
| black | black の場合1 |
| white | white の場合1 |
| ctrmgpa | trmgpaの変化 |
| ctothrs | total hoursの変化 |
| ccrsgpa | crsgpaの変化 |
| ccrspop | crspopの変化 |
| cseason | seasonの変化 |
| hsperc | 高校クラス順位（%） |
| football | football player の場合1 |
