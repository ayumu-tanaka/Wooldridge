# GPA3 データセットの説明

**観測数（Obs）:** 732

## 変数一覧

| 変数名 | 説明 |
|------|------|
| term | fall = 1, spring = 2 |
| sat | SAT score |
| tothrs | total hours prior to term |
| cumgpa | cumulative GPA |
| season | in season の場合1 |
| frstsem | student's 1st semester の場合1 |
| crsgpa | weighted course GPA |
| verbmath | verbal SAT to math SAT ratio |
| trmgpa | term GPA |
| hssize | size high school graduating class |
| hsrank | rank in h.s. class |
| id | student identifier |
| spring | spring term の場合1 |
| female | 女性の場合1 |
| black | 黒人の場合1 |
| white | 白人の場合1 |
| ctrmgpa | trmgpaの変化 |
| ctothrs | total hoursの変化 |
| ccrsgpa | crsgpaの変化 |
| ccrspop | crspopの変化 |
| cseason | seasonの変化 |
| hsperc | 100*(rank/hssize) |
| football | football player の場合1 |
