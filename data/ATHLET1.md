# ATHLET1 データセットの説明

**観測数（Obs）:** 118

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1992または1993年 |
| apps | 入学願書数 |
| top25 | 新入生のうち高校上位25%に入る割合 |
| ver500 | 新入生のうちSATの口語500以上の割合 |
| mth500 | 新入生のうちSATの数学500以上の割合 |
| stufac | 学生教員比率 |
| bowl | bowl game in prev year の場合1 |
| btitle | men's cnf chmps prev year の場合1 |
| finfour | men's final 4 prev year の場合1 |
| lapps | appsの対数 |
| d93 | year = 1993 の場合1 |
| avg500 | SAT平均点（口語・数学） |
| cfinfour | finfourの変化 |
| clapps | lappsの変化 |
| cstufac | stufacの変化 |
| cbowl | bowlの変化 |
| cavg500 | avg500の変化 |
| cbtitle | btitleの変化 |
| lapps_1 | lappsのラグ |
| school | 大学名 |
| ctop25 | top25の変化 |
| bball | btitle or finfour の場合1 |
| cbball | bballの変化 |
