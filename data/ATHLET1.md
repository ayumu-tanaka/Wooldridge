# ATHLET1 データセットの説明

**観測数（Obs）:** 118

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1992 or 1993 |
| apps | 入学願書数 |
| top25 | 新入生のうち高校上位25%に入る割合 |
| ver500 | 新入生のうちSATの口語500以上の割合 |
| mth500 | 新入生のうちSATの数学500以上の割合 |
| stufac | 学生教員比率 |
| bowl | 前年度ボウルゲーム出場の場合1 |
| btitle | 前年度男子カンファレンスチャンピオンの場合1 |
| finfour | 前年度男子ファイナル4進出の場合1 |
| lapps | 願書数の対数 |
| d93 | 1993年の場合1 |
| avg500 | SAT平均点（口語・数学） |
| cfinfour | finfourの変化 |
| clapps | lappsの変化 |
| cstufac | stufacの変化 |
| cbowl | bowlの変化 |
| cavg500 | avg500の変化 |
| cbtitle | btitleの変化 |
| lapps_1 | lappsのラグ |
| school | 大学名 |
| ctop25 | top25の変化 |
| bball | btitleまたはfinfourの場合1 |
| cbball | bballの変化 |
