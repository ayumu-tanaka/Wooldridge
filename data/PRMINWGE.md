# PRMINWGE データセットの説明

**観測数（Obs）:** 38

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1950-1987 |
| avgmin | weighted avg min wage, 44 indust |
| avgwage | wghted avg hourly wage, 44 indust |
| kaitz | Kaitz min wage index |
| avgcov | wghted avg coverage, 8 industries |
| covt | economy-wide coverage of min wage |
| mfgwage | avg manuf. wage |
| prdef | Puerto Rican price deflator |
| prepop | PR employ/popul ratio |
| prepopf | PR employ/popul ratio, alternative |
| prgnp | PR GNP |
| prunemp | PR unemployment rate |
| usgnp | US GNP |
| t | time trend:  1 to 38 |
| post74 | time trend:  starts in 1974 |
| lprunemp | prunempの対数 |
| lprgnp | prgnpの対数 |
| lusgnp | usgnpの対数 |
| lkaitz | kaitzの対数 |
| lprun_1 | lprunemp[t-1] |
| lprepop | prepopの対数 |
| lprep_1 | lprepop[t-1] |
| mincov | (avgmin/avgwage)*avgcov |
| lmincov | mincovの対数 |
| lavgmin | avgminの対数 |
