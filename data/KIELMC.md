# KIELMC データセットの説明

**観測数（Obs）:** 321

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1978 or 1981 |
| age | age of house |
| agesq | 年齢の二乗 |
| nbh | neighborhood #, 1 to 6 |
| cbd | dist. to central bus. dstrct, feet |
| intst | 州間高速道路までの距離（フィート） |
| lintst | intstの対数 |
| price | selling price |
| rooms | 住宅の部屋数 |
| area | square footage of house |
| land | square footage lot |
| baths | 浴室数 |
| dist | dist. from house to incinerator, feet |
| ldist | 距離の対数 |
| wind | perc. time wind incin. to house |
| lprice | 価格の対数 |
| y81 | year == 1981 の場合1 |
| larea | 面積の対数 |
| lland | landの対数 |
| y81ldist | y81*ldist |
| lintstsq | lintstの二乗 |
| nearinc | dist <= 15840 の場合1 |
| y81nrinc | y81*nearinc |
| rprice | price, 1978 dollars |
| lrprice | rpriceの対数 |
