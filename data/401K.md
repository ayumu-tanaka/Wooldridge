# 401K データセットの説明

**観測数（Obs）:** 1534

## 変数一覧

| 変数名 | 説明 |
|------|------|
| prate | 参加率（%） |
| mrate | 401kプランのマッチ率 |
| totpart | 401k参加者の総数 |
| totelg | 401kプランの加入資格者の総数 |
| age | 401kプランの年数 |
| totemp | 企業の総従業員数 |
| sole | 401k is firm's sole plan の場合1 |
| ltotemp | totempの対数 |
