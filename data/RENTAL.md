# RENTAL データセットの説明

**観測数（Obs）:** 128

## 変数一覧

| 変数名 | 説明 |
|------|------|
| city | city label, 1 to 64 |
| year | 80 or 90 |
| pop | city population |
| enroll | # college students enrolled |
| rent | average rent |
| rnthsg | renter occupied units |
| tothsg | occupied housing units |
| avginc | 一人当たり所得 |
| lenroll | enrollの対数 |
| lpop | 人口の対数 |
| lrent | rentの対数 |
| ltothsg | tothsgの対数 |
| lrnthsg | rnthsgの対数 |
| lavginc | avgincの対数 |
| clenroll | lrent from 80 to 90の変化 |
| clpop | lpop（人口対数）の変化 |
| clrent | lrentの変化 |
| cltothsg | ltothsgの変化 |
| clrnthsg | lrnthsgの変化 |
| clavginc | lavgincの変化 |
| pctstu | percent of population students |
| cpctstu | pctstuの変化 |
| y90 | year == 90 の場合1 |

* * *
