# RENTAL データセットの説明

**観測数（Obs）:** 128

## 変数一覧

| 変数名 | 説明 |
|------|------|
| city | 都市ラベル（1〜64） |
| year | 80または90年 |
| pop | 都市人口 |
| enroll | 大学生在籍数 |
| rent | 平均家賃 |
| rnthsg | 賃貸住宅戸数 |
| tothsg | 居住住宅戸数 |
| avginc | 一人当たり所得 |
| lenroll | enrollの対数 |
| lpop | popの対数 |
| lrent | rentの対数 |
| ltothsg | tothsgの対数 |
| lrnthsg | rnthsgの対数 |
| lavginc | avgincの対数 |
| clenroll | lrent from 80 to 90の変化 |
| clpop | lpopの変化 |
| clrent | lrentの変化 |
| cltothsg | ltothsgの変化 |
| clrnthsg | lrnthsgの変化 |
| clavginc | lavgincの変化 |
| pctstu | 学生人口割合（%） |
| cpctstu | pctstuの変化 |
| y90 | year == 90 の場合1 |
