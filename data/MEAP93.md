# MEAP93 データセットの説明

**観測数（Obs）:** 408

## 変数一覧

| 変数名 | 説明 |
|------|------|
| lnchprg | 給食プログラムに参加する生徒の割合（%） |
| enroll | 学校在籍数 |
| staff | 生徒1000人当たりのスタッフ数 |
| expend | 生徒一人当たりの支出（ドル） |
| salary | 教員平均給与（ドル） |
| benefits | 教員平均福利厚生（ドル） |
| droprate | 退学率（%） |
| gradrate | 卒業率（%） |
| math10 | MEAP数学に合格した生徒の割合（%） |
| sci11 | MEAP理科に合格した生徒の割合（%） |
| totcomp | 給与＋福利厚生 |
| ltotcomp | totcompの対数 |
| lexpend | 支出の対数 |
| lenroll | enrollの対数 |
| lstaff | staffの対数 |
| bensal | 給付÷給与 |
| lsalary | salaryの対数 |
