# BWGHT データセットの説明

| 変数名 | 説明 |

| faminc | 家族収入（千ドル、1988年） |
| cigtax | 居住州のタバコ税（1988年） |
| cigprice | 居住州のタバコ価格（1988年） |

| fatheduc | 父親の就学年数 |
| motheduc | 母親の就学年数 |
| parity | 子供の出生順位 |
| male | 男児の場合1 |
| white | 白人の場合1 |
| cigs | 妊娠中の1日あたり喫煙本数 |
| lbwght | 出生体重の対数 |
| bwghtlbs | 出生体重（ポンド） |
| packs | 妊娠中の1日あたり喫煙パック数 |
| lfaminc | 家族収入の対数 |
