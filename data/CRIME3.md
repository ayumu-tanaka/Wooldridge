# CRIME3 データセットの説明

**観測数（Obs）:** 106

## 変数一覧

| 変数名 | 説明 |
|------|------|
| district | 地区番号 |
| year | 72または78（年） |
| crime | 1000人あたりの犯罪件数 |
| clrprc1 | 解決率（前年度） |
| clrprc2 | 解決率（2年前） |
| d78 | 1978年の場合1 |
| avgclr | clrprc1とclrprc2の平均 |
| lcrime | 犯罪数の対数 |
| clcrime | lcrime（犯罪数対数）の変化 |
| cavgclr | avgclrの変化 |
| cclrprc1 | clrprc1の変化 |
| cclrprc2 | clrprc2の変化 |
