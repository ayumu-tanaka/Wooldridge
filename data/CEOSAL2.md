# CEOSAL2 データセットの説明

**観測数（Obs）:** 177

## 変数一覧

| 変数名 | 説明 |
|------|------|
| salary | 報酬（千ドル、1990年） |
| age | 年齢（年） |
| college | attended college の場合1 |
| grad | attended graduate school の場合1 |
| comten | 在社年数 |
| ceoten | CEO在任年数 |
| sales | 企業売上高（百万ドル、1990年） |
| profits | 利益（百万ドル、1990年） |
| mktval | 時価総額（百万ドル、1990年末） |
| lsalary | salaryの対数 |
| lsales | salesの対数 |
| lmktval | mktvalの対数 |
| comtensq | comtenの二乗 |
| ceotensq | ceotenの二乗 |
| profmarg | 売上高利益率（%） |
