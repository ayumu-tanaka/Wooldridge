# VOTE1 データセットの説明

**観測数（Obs）:** 173

## 変数一覧

| 変数名 | 説明 |
|------|------|
| state | 州の郵便番号 |
| district | 議会選挙区 |
| democA | A is democrat の場合1 |
| voteA | 候補者Aへの投票率（%） |
| expendA | 候補者Aの選挙費用（千ドル） |
| expendB | 候補者Bの選挙費用（千ドル） |
| prtystrA | 大統領への投票率（%） |
| lexpendA | expendAの対数 |
| lexpendB | expendBの対数 |
| shareA | 候補者A費用割合（%） |
