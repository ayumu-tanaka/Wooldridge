# VOTE1 データセットの説明

**観測数（Obs）:** 173

## 変数一覧

| 変数名 | 説明 |
|------|------|
| state | 州の郵便番号 |
| district | congressional district |
| democA | A is democrat の場合1 |
| voteA | percent vote for A |
| expendA | campaign expends. by A, $1000s |
| expendB | campaign expends. by B, $1000s |
| prtystrA | 大統領への投票率（%） |
| lexpendA | 候補者A選挙費用の対数 |
| lexpendB | 候補者B選挙費用の対数 |
| shareA | 候補者Aの費用割合（%） |
