# INTDEF データセットの説明

**観測数（Obs）:** 49

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1948〜1996年 |
| i3 | 3ヶ月T-bill金利 |
| inf | CPI（消費者物価指数）インフレ率 |
| rec | 連邦収入（GDP比、%） |
| out | 連邦支出（GDP比、%） |
| def | 財政収支（GDP比、%） |
| i3_1 | 3ヶ月金利（1期ラグ） |
| inf_1 | インフレ率（1期ラグ） |
| def_1 | 財政赤字（1期ラグ） |
| ci3 | 3ヶ月金利の変化 |
| cinf | インフレ率の変化 |
| cdef | 財政赤字の変化 |
| y77 | 1977年以降の場合1（会計年度変更） |
