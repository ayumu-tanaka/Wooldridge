| 変数名 | 説明 |
|------|------|
| educ | 就学年数 |
| south | live in south の場合1 |
| nonwhite | nonwhite の場合1 |
| female | female の場合1 |
| married | married の場合1 |
| exper | 経験年数（年齢 - 就学年数 - 6） |
| expersq | experの二乗 |
| union | belong to union の場合1 |
| lwage | 時給の対数 |

| year | 78または85年 |
| y85 | year == 85 の場合1 |
| y85fem | y85 × female（交差項） |
| y85educ | y85 × educ（交差項） |
| y85union | y85 × union（交差項） |
