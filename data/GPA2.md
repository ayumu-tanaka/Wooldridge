# GPA2 データセットの説明

**観測数（Obs）:** 4137

## 変数一覧

| 変数名 | 説明 |
|------|------|
| sat | SAT合計スコア |
| tothrs | 秋学期を通じた総単位時間 |
| colgpa | 秋学期後のGPA |
| athlete | athlete の場合1 |
| verbmath | SAT口語/数学スコア |
| hsize | 卒業クラスの規模（百人） |
| hsrank | 卒業クラスでの順位 |
| hsperc | 高校卒業クラスでの順位（%） |
| female | female の場合1 |
| white | white の場合1 |
| black | black の場合1 |
| hsizesq | hsizeの二乗 |
