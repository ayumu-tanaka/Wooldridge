| rptchall | a repeat challenger の場合1 |
| tenure | HR部門の勤続年数 |
| lawyer | law degree の場合1 |
| linexp90 | inexp90の対数 |
| lchexp90 | chexp90の対数 |
| linexp88 | inexp88の対数 |
| lchexp88 | chexp88の対数 |
| incshr90 | 現職者支出割合（%、1990年） |
