# HPRICE3 データセットの説明

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1978, 1981 |
| age | age of house |
| agesq | 年齢の二乗 |
| nbh | neighborhood, 1 to 6 |
| cbd | dist. to central bus. dstrct, feet |
| inst | 州間高速道路までの距離（フィート） |
| linst | instの対数 |
| price | selling price |
| rooms | 住宅の部屋数 |
| area | square footage of house |
| land | square footage lot |
| baths | 浴室数 |
| dist | dist. from house to incin., feet |
| ldist | 距離の対数 |
| lprice | 価格の対数 |
| y81 | year = 1981 の場合1 |
| larea | 面積の対数 |
| lland | landの対数 |
| linstsq | linstの二乗 |
