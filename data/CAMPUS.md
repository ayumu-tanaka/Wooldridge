# CAMPUS データセットの説明

**観測数（Obs）:** 97

## 変数一覧

| 変数名 | 説明 |
|------|------|
| enroll | 総在籍者数 |
| priv | private college の場合1 |
| police | 在籍警察官数 |
| crime | キャンパス内の総犯罪件数 |
| lcrime | crimeの対数 |
| lenroll | enrollの対数 |
| lpolice | policeの対数 |
