## 変数一覧

| 変数名 | 説明 |
|------|------|
| id | 識別子 |
| male | 男性の場合 |
| age | 年数 |
| yrsmarr | 既婚ダミー |

| relig | 宗教度（5=非常に信仰、4=まあまあ、3=少し、2=ほとんどなし、1=反対） |
| educ | 就学年数 |

| ratemarr | 婚姻満足度（5=非常に幸福、4=平均以上、3=平均、2=やや不幸、1=非常に不幸） |
| naffairs | 過去1年間の情事の件数 |
| affair | 少なくとも1件の情事があった場合 |
| vryhap | 5の場合 |
| hapavg | 4の場合 |
| avgmarr | 3の場合 |
