# SMOKE データセットの説明

**観測数（Obs）:** 807

## 変数一覧

| 変数名 | 説明 |
|------|------|
| educ | 就学年数 |
| cigpric | 州のタバコ価格（セント/パック） |
| white | white の場合1 |
| age | 年齢（年） |
| income | 年収（ドル） |
| cigs | 1日あたりの喫煙本数 |
| restaurn | state restaurant smoking restrictions の場合1 |
| lincome | incomeの対数 |
| agesq | ageの二乗 |
| lcigpric | cigpriceの対数 |
