# OPENNESS データセットの説明

**観測数（Obs）:** 114

## 変数一覧

| 変数名 | 説明 |
|------|------|
| open | 輸入GDP比（%、1973年以降） |
| inf | 年平均インフレ率（1973年以降） |
| pcinc | 一人当たり所得（1980年、米ドル） |
| land | 土地面積（平方マイル） |
| oil | major oil producer の場合1 |
| good | 'good' data の場合1 |
| lpcinc | pcincの対数 |
| lland | landの対数 |
| lopen | openの対数 |
| linf | infの対数 |
| opendec | 営業時間÷100 |
| linfdec | inf/100の対数 |
