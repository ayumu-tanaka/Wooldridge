# INFMRT データセットの説明

**観測数（Obs）:** 102

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1987 or 1990 |
| infmort | infant mortality rate |
| afdcprt | AFDC participation, 1000s |
| popul | population, 1000s |
| pcinc | 一人当たり所得 |
| physic | Drs. per 100,000 civilian pop. |
| afdcper | percent on AFDC |
| d90 | year == 1990 の場合1 |
| lpcinc | 一人当たり所得の対数 |
| lphysic | physicの対数 |
| DC | =1 for Washington DC |
| lpopul | populの対数 |
