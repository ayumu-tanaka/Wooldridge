# INFMRT データセットの説明

**観測数（Obs）:** 102

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1987または1990年 |
| infmort | 乳児死亡率 |
| afdcprt | AFDC参加数（千） |
| popul | 人口（千人） |
| pcinc | 一人当たり所得 |
| physic | 民間人口10万人当たりの医師数 |
| afdcper | AFDC参加割合（%） |
| d90 | year == 1990 の場合1 |
| lpcinc | pcincの対数 |
| lphysic | physicの対数 |
| DC | ワシントンDCの場合1 |
| lpopul | populの対数 |
