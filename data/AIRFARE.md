# AIRFARE データセットの説明

**観測数（Obs）:** 4596

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1998、1999、2000年 |
| origin | 出発地 |
| destin | 目的地 |
| id | 識別子 |
| dist | マイル単位 |
| passen | 1日当たりの乗客数 |
| fare | 片道運賃（ドル） |
| bmktshr | 最大航空会社の市場 |
| ldist | distanceの対数 |
| y98 | 1998年の場合 |
| y99 | 1999年の場合 |
| y00 | 2000年の場合 |
| lfare | fareの対数 |
| ldistsq | ldistの二乗 |
| concen | 市場シェア |
| lpassen | passenの対数 |
