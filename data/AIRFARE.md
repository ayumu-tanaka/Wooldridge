# AIRFARE データセットの説明

**観測数（Obs）:** 4596

## 変数一覧

| 変数名 | 説明 |
|------|------|
| year | 1998, 1999, 2000 |
| origin | origin |
| destin | destination |
| id | 識別子 |
| dist | in miles |
| passen | passengers per day |
| fare | one-way fare, $ |
| bmktshr | market, biggest carrier |
| ldist | distanceの対数 |
| y98 | if year == 1998 |
| y99 | if year == 1999 |
| y00 | if year == 2000 |
| lfare | fareの対数 |
| ldistsq | ldistの二乗 |
| concen | bmktshr |
| lpassen | passenの対数 |

* * *
