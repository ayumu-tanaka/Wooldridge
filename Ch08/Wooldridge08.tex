\section{8-1
不均一分散の含意}\label{ux4e0dux5747ux4e00ux5206ux6563ux306eux542bux610f}

\begin{frame}{不均一分散 (Heteroskedasticity) とは}
\phantomsection\label{ux4e0dux5747ux4e00ux5206ux6563-heteroskedasticity-ux3068ux306f}
\begin{itemize}
\tightlist
\item
  これまでのガウス＝マルコフ仮定（MLR.5）では、説明変数 \(x\)
  の値に関わらず、誤差項 \(u\) の分散が一定であることを仮定していた。
\item
  \(Var(u|x) = \sigma^2\) （等分散性, Homoskedasticity）
\item
  これが満たされない場合（\(Var(u|x) = \sigma^2(x)\)）、\textbf{不均一分散}と呼ぶ。
\end{itemize}
\end{frame}

\begin{frame}{不均一分散がOLSに与える影響}
\phantomsection\label{ux4e0dux5747ux4e00ux5206ux6563ux304colsux306bux4e0eux3048ux308bux5f71ux97ff}
\begin{enumerate}
\tightlist
\item
  \textbf{不偏性・一致性には影響しない。}
  OLS推定量は依然として不偏かつ一致である。
\item
  \textbf{OLSはもはや最良線形不偏推定量 (BLUE) ではない。}
  より効率的な推定量が存在し得る。
\item
  \textbf{標準誤差の推定が正しくなくなる。} 通常の \(t\) 統計量や \(F\)
  統計量を用いた仮説検定が妥当ではなくなる。
\end{enumerate}
\end{frame}

\section{8-2
不均一分散に頑健な推論}\label{ux4e0dux5747ux4e00ux5206ux6563ux306bux9811ux5065ux306aux63a8ux8ad6}

\begin{frame}{頑健な標準誤差 (Robust Standard Errors)}
\phantomsection\label{ux9811ux5065ux306aux6a19ux6e96ux8aa4ux5dee-robust-standard-errors}
\begin{itemize}
\tightlist
\item
  不均一分散が存在しても、適切な修正を加えることで妥当な推論（\(t\)
  検定など）が可能になる。
\item
  ホワイトの頑健な標準誤差（White standard
  errors）や、不均一分散に頑健な標準誤差（Heteroskedasticity-robust
  standard errors）と呼ばれる。
\item
  大規模標本において、誤差項の分散の形を知らなくても正当化される。
\end{itemize}
\end{frame}

\section{8-3
不均一分散の検定}\label{ux4e0dux5747ux4e00ux5206ux6563ux306eux691cux5b9a}

\begin{frame}{なぜ検定するのか？}
\phantomsection\label{ux306aux305cux691cux5b9aux3059ux308bux306eux304b}
\begin{itemize}
\tightlist
\item
  不均一分散が疑われる場合、まず統計的に確認することが重要。
\item
  \textbf{検定の役割：}
  「等分散という帰無仮説を棄却できるか」を判断する。
\item
  主な検定方法：

  \begin{itemize}
  \tightlist
  \item
    \textbf{ブロイシュ＝ペーガン (BP)
    検定}：分散が説明変数の線形関数である場合に強力。
  \item
    \textbf{ホワイト (White)
    検定}：より一般的な不均一分散（非線形など）も検出できる。
  \end{itemize}
\item
  \(p\)
  値が小さい（例：0.05未満）なら、等分散の仮定を棄却し、頑健な標準誤差やWLSで対応する。
\end{itemize}
\end{frame}

\begin{frame}{ブロイシュ＝ペーガン検定 (Breusch-Pagan Test)}
\phantomsection\label{ux30d6ux30edux30a4ux30b7ux30e5ux30daux30fcux30acux30f3ux691cux5b9a-breusch-pagan-test}
\begin{itemize}
\tightlist
\item
  \textbf{帰無仮説}
  \(H_0: Var(u|x_1, \dots, x_k) = \sigma^2\)（等分散、不均一分散なし）
\item
  \textbf{対立仮説} \(H_1\)：分散が少なくとも1つの \(x_j\)
  に依存する（不均一分散あり）
\item
  \textbf{直感：} もし分散が \(x_j\) に無関係なら、\(\hat{u}^2\) を
  \(x_j\) で回帰しても \(R^2 \approx 0\) になるはず。
\item
  \textbf{手順：}

  \begin{enumerate}
  \tightlist
  \item
    OLS残差 \(\hat{u}\) を求める。
  \item
    残差の2乗 \(\hat{u}^2\) を独立変数 \(x_1, \dots, x_k\)
    で\textbf{補助回帰}する。
  \item
    \(LM = n \cdot R^2\) を計算し、自由度 \(k\) の \(\chi^2\) 分布で
    \(p\) 値を求める。
  \end{enumerate}
\end{itemize}
\end{frame}

\begin{frame}{ホワイト検定 (White Test)}
\phantomsection\label{ux30dbux30efux30a4ux30c8ux691cux5b9a-white-test}
\begin{itemize}
\tightlist
\item
  \textbf{帰無仮説} \(H_0\)：等分散（BP検定と同じ）
\item
  \textbf{BP検定との違い：}
  補助回帰に説明変数の\textbf{2乗項・交差項}も含める。

  \begin{itemize}
  \tightlist
  \item
    分散が非線形な形で \(x_j\) に依存する場合も検出できる。
  \end{itemize}
\item
  \textbf{問題点：} 変数が多いと交差項が膨大になり自由度が急減する。
\item
  \textbf{簡略版（推奨）：} OLS予測値 \(\hat{y}\) とその2乗
  \(\hat{y}^2\) を補助回帰に使う。

  \begin{itemize}
  \tightlist
  \item
    \(\hat{y}\)
    は全説明変数の線形結合なので、2乗項・交差項の情報を集約している。
  \item
    自由度の消費は常に2（\(\hat{y}\) と \(\hat{y}^2\) の2変数）。
  \end{itemize}
\end{itemize}
\end{frame}

\section{8-4 加重最小二乗法
(WLS)}\label{ux52a0ux91cdux6700ux5c0fux4e8cux4e57ux6cd5-wls}

\begin{frame}[fragile]{加重最小二乗法 (Weighted Least Squares)}
\phantomsection\label{ux52a0ux91cdux6700ux5c0fux4e8cux4e57ux6cd5-weighted-least-squares}
\begin{itemize}
\tightlist
\item
  不均一分散の構造 \(Var(u|\mathbf{x}) = \sigma^2 h(\mathbf{x})\)
  が既知である場合、OLSよりも効率的な推定が可能。
\item
  各観測値を \(1/\sqrt{h_i}\) で割って変換し、変換モデルにOLSを適用 →
  これがWLS。
\item
  Rでは \texttt{lm()} の \texttt{weights} 引数に \(1/h_i\)
  を指定するだけでよい。
\item
  これを\textbf{一般化最小二乗法 (GLS)} の一種と呼ぶ。
\end{itemize}
\end{frame}

\begin{frame}[fragile]{WLS の実装例：Example 8.6（金融資産方程式）}
\phantomsection\label{wls-ux306eux5b9fux88c5ux4f8bexample-8.6ux91d1ux878dux8cc7ux7523ux65b9ux7a0bux5f0f}
\begin{itemize}
\tightlist
\item
  データ：\texttt{k401ksubs}（単身世帯 \texttt{fsize\ ==\ 1}）
\item

\begin{frame}{実行可能な一般化最小二乗法 (FGLS)}
\phantomsection\label{ux5b9fux884cux53efux80fdux306aux4e00ux822cux5316ux6700ux5c0fux4e8cux4e57ux6cd5-fgls}
\begin{itemize}
\tightlist
\item
  \(h(\mathbf{x})\) の形が未知の場合、データから推定する →
  \textbf{FGLS（実行可能GLS）}。
\item
  分散関数のモデル化：\(Var(u|\mathbf{x}) = \sigma^2 \exp(\delta_0 + \delta_1 x_1 + \cdots + \delta_k x_k)\)

  \begin{itemize}
  \tightlist
  \item
    指数関数を使うので推定された分散は常に正。
  \end{itemize}
\item
  \textbf{FGLS の手順：}

  \begin{enumerate}
  \tightlist
  \item
    OLSを実行し残差 \(\hat{u}\) を取得。
  \item
    \(\log(\hat{u}^2)\) を独立変数で回帰し、予測値 \(\hat{g}_i\)
    を得る。
  \item
    \(\hat{h}_i = \exp(\hat{g}_i)\) を計算（分散の推定値）。
  \item
    重み \(1/\hat{h}_i\) でWLSを実行。
  \end{enumerate}
\end{itemize}
\end{frame}

\begin{frame}{まとめ}
\phantomsection\label{ux307eux3068ux3081}
\begin{itemize}
\tightlist
\item
  不均一分散下では、OLS推定量は不偏だが効率的ではなく、通常の検定が使えない。
\item
  \textbf{解決策1:} 頑健な標準誤差を用いる（最も一般的）。
\item
  \textbf{解決策2:}
  不均一分散の検定を行い、必要に応じてWLS/FGLSで効率性を高める。
\item
  実証分析では、まずは頑健な標準誤差で結果を確認するのが定石である。
\end{itemize}
\end{frame}
